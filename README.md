# SomaOS — Not Another LLM.

Last year, while everyone was chasing bigger models, we asked: the human brain doesn't work by stacking parameters. Why should AI?

We took a different path.

## What We Built

An AI system with 19 brain regions modeled after human neuroanatomy. Not a metaphor. Not prompt engineering. Real functional modules — brainstem, thalamus, amygdala, hippocampus, cerebral cortex, prefrontal cortex, cerebellum — each running independently, communicating through a corpus callosum, competing for "consciousness" in a Global Workspace.

It's called SomaOS.

## Three Things

**1. Architecture grounded in neuroscience.**
A state-of-the-art 19-region brain architecture processing memory consolidation from hippocampus to cortex. Every component backed by peer-reviewed neuroscience.

**2. It evolves itself.**
Not "human tunes parameters and retrains." It analyzes its own brain region source code, proposes improvements, tests them in a sandbox, writes to disk, and hot-reloads. Broken changes roll back automatically. Three agents run 24/7, each continuously rewriting itself.

**3. Verifiable results.**
General cognition, engineering, and legal agents — each with hundreds of evolution records, brain region source code modified dozens of times, consciousness focus shifting with dopamine levels.

We're not building a bigger model. We're building AI with instincts, memory, and the ability to improve itself.
